# 
WIP Uses openai-whisper to transcribe audio to text locally.

## App Deps
```
pip install numpy
pip install openai-whisper
pip install sounddevice
pip install rich
pip install streamlit
```

Works with Python 3.12 but not with 3.13 atm.
